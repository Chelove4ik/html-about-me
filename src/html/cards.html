Эль Примо (англ. El Primo) – Редкий боец предназначенный для ближнего боя класса "Тяжеловес". Его можно получить из любого ящика.

Леон (англ. Leon) — легендарный боец класса "Скрытный убийца". Его можно получить из любого ящика.

Шелли (англ. Shelly) — начальный боец класса «Воин», который даётся каждому игроку в начале игры.
